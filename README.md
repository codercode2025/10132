# 基于Web的学习辅助系统

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=10132

## 技术介绍
Java语言 + ssm框架/springboot框架（后端）+ Vue前端框架（Web）+ Mysql(数据库)

## 开发工具
IDEA或者MyEclipse + Navicat(mysql数据库可视化工具) + Tomcat

## 项目视频演示
- [演示视频1](https://graduation-images.oss-cn-beijing.aliyuncs.com/videos/70%E5%A5%97-2-ssm%E5%BD%95%E5%83%8F/10132_ssm%E4%B8%89%E7%9C%81%E5%AD%A6%E5%A0%82%E2%80%94%E5%AD%A6%E4%B9%A0%E8%BE%85%E5%8A%A9%E7%B3%BB%E7%BB%9F%E6%BC%94%E7%A4%BA%E5%BD%95%E5%83%8F2022.mp4)

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
